# 斗地主游戏逻辑模拟

## 项目描述

这是一个本地的斗地主游戏逻辑算法模拟项目。项目实现了斗地主游戏的核心逻辑，包括牌的创建、洗牌、分发、玩家管理和游戏规则判断等功能。通过这个项目，您可以模拟斗地主游戏的完整流程，而无需实际的网络连接或用户界面。

## 主要特点

- **本地模拟**：项目运行在本地，无需网络连接，完全模拟斗地主游戏逻辑。
- **算法实现**：包含完整的斗地主游戏算法，包括牌型判断、胜负判定等。
- **多语言支持**：项目带有中文语言包JSON文件（lan_ch.json），支持多语言扩展。
- **可扩展性**：如果接入网络和UI组件，可以成为一个完整可玩的斗地主游戏。

## 安装要求

- Python 3.x
- 无需安装第三方包（所有依赖均为Python标准库）

## 运行方法

从项目根目录运行以下命令：

```bash
python3 server.py
```

## 多语言支持

项目使用JSON文件存储语言包，目前包含中文语言包（lan_ch.json）。您可以通过修改或添加新的JSON文件来支持其他语言。

## 扩展性

这个项目是一个纯逻辑模拟，如果您想将其扩展为完整游戏，可以：

- 添加网络模块实现多人在线对战
- 集成图形用户界面（GUI）
- 添加更多语言包
